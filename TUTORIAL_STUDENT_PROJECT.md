# 📚 Comment publier un projet sur Inside Eikon

Bienvenue ! Ce guide vous montrera comment créer et éditer votre projet sur Inside Eikon. Nous explorerons chaque section de la page d'édition étape par étape.

---

## 🔐 Avant de commencer : Se connecter

1. Allez sur **https://inside.eikon.ch/wp-admin**
2. Entrez vos identifiants :
   - **Nom d'utilisateur** : votre adresse email ou nom d'utilisateur
   - **Mot de passe** : votre mot de passe personnel
3. Cliquez sur **Se connecter**

> ⚠️ **Conseil** : Si vous avez oublié votre mot de passe, cliquez sur « Mot de passe perdu ? » et suivez les instructions.

---

## ✏️ Créer un nouveau projet

1. Dans le menu de gauche, cliquez sur **Projets**
2. Cliquez sur le bouton bleu **+ Ajouter** (ou **Ajouter un nouveau projet**)
3. Vous arrivez sur la page d'édition avec plusieurs sections

---

## 📝 Section 1 : TITRE

**Où c'est ?** : En haut de la page

Le titre est le premier élément que les visiteurs voient. Soyez explicite et descriptif.

### Conseils pour un bon titre :

- Soyez **clair et spécifique** : "Typographie pour affiche de concert" plutôt que "Mon projet"
- Utilisez **2-5 mots** : assez courts pour être impactants
- **Décrivez le sujet** : que fait le projet ? quel est le contexte ?
- Évitez les caractères spéciaux ou accents excessifs

### Exemples de bons titres :

- "Redesign d'identité visuelle pour startup"
- "Série photographique : Les textures urbaines"
- "Motion design d'une publicité 15 secondes"
- "Typographie expérimentale avec lettres cassées"

---

## 📄 Section 2 : EXTRAIT

**Où c'est ?** : Sous le titre, dans une petite boîte grise

L'extrait est un **court résumé** qui apparaît parfois dans les listes et sur la page de détail. C'est votre "accroche" pour donner envie de voir plus.

### Ce qu'il faut y mettre :

- Une **phrase courte** (20-50 mots)
- Le **contexte** : pour quel cours ou projet ?
- L'**objectif principal** : qu'avez-vous cherché à explorer ?
- L'**approche** : votre angle personnel

### Exemple d'extrait :

> Exploration typographique expérimentale utilisant des lettres décalées pour créer du dynamisme. Ce projet teste les limites entre lisibilité et esthétique.

---

## 🎨 Section 3 : VIGNETTE (Featured Image)

**Où c'est ?** : Sous l'extrait ou en haut à gauche de la page

La vignette est l'**image de couverture** - celle qu'on voit en premier sur le site. C'est crucial pour l'impact visuel.

### Comment ajouter une vignette :

1. Cliquez sur **Ajouter une image à la une** ou sur la zone grise
2. Vous pouvez :
   - **Charger une image** depuis votre ordinateur (JPG, PNG, WebP)
   - **Choisir une image existante** de la médiathèque
3. Sélectionnez votre image et cliquez **Utiliser comme image à la une**

### Conseils pour une bonne vignette :

- **Qualité élevée** : au moins 1200x800px
- **Représentative** : elle doit montrer l'essence de votre projet
- **Bien composée** : utilisez les 1/3 et points d'intérêt visuels
- **Contraste** : assure la lisibilité même en petit
- **Couleurs** : choisissez une image avec des couleurs attrayantes

> 💡 **Astuce** : Prenez une capture d'écran de votre meilleur travail, ou une photo bien cadrée de votre création.

---

## 📝 Section 4 : CONTENU (Éditeur de texte)

**Où c'est ?** : Zone principale de la page

Le contenu principal est votre **description détaillée**. Elle explique le contexte, votre processus et vos réflexions.

### Ce qu'il faut y inclure :

- **Le contexte** : pour quel cours ? quelle brief ? quel problème à résoudre ?
- **Votre approche** : comment avez-vous commencé ? quels sont vos choix ?
- **Les enjeux** : quelles étaient les contraintes ? les défis ?
- **Votre réflexion** : pourquoi ces décisions ? qu'avez-vous appris ?
- **Les résultats** : comment jugez-vous le projet ? succès ou points d'amélioration ?

### Outils de formatage disponibles :

- **Gras** (B) : pour mettre en avant des mots-clés
- **Italique** (I) : pour les citations ou nuances

### Exemple de contenu :

> Typographie pour une affiche de concert. Ce projet explore l'utilisation de caractères décalés pour créer dynamisme et énergie. J'ai travaillé sur la hiérarchie visuelle et la lisibilité tout en conservant une esthétique expérimentale. Le défi était de maintenir la légibilité tout en cassant les conventions typographiques. Le résultat final montre un bon équilibre entre innovation et clarté.

---

## 🖼️ Section 5 : GALERIE (Projet)

**Où c'est ?** : Après le contenu, dans une section "Projet"

La galerie est le **cœur visuel** de votre portfolio. Elle montre votre travail avec différents types de contenu mélangés.

### Types de contenu disponibles :

#### 📸 IMAGE/PHOTO

- Cliquez sur **Charger une image** ou sélectionnez dans la médiathèque
- Vous pouvez ajouter un **texte descriptif** (optionnel)
- Bon pour : mockups, résultats finals, captures d'écran, photos

#### 📝 TEXTE

- Permet d'ajouter des explications, citations, ou process
- Utilisez le **gras** et l'**italique** pour mettre en avant
- Bon pour : explications de processus, citations inspirantes, réflexions

#### 🎬 VIDÉO

- Sélectionnez une vidéo uploadée dans la médiathèque
- Affiche une miniature et un lecteur de vidéo
- Bon pour : animations, tests, présentation du résultat en mouvement

#### 🖼️ GALERIE

- Permet de grouper plusieurs images ensemble
- Elles s'affichent en grille
- Bon pour : variations, itérations, processus étape par étape, mockups

### Comment organiser votre galerie :

1. **Racontez une histoire** : ne mettez pas les images dans l'ordre aléatoire
2. **Mélangez les formats** : alternez images, textes, vidéos
3. **Commencez fort** : mettez votre meilleur travail en premier
4. **Montrez le processus** : quelques étapes intermédiaires
5. **Terminez avec le résultat** : l'image ou vidéo finale

### Exemple d'organisation :

Image résultat → Texte expliquant la brief → Galerie des variations → Vidéo animée → Texte sur les retours → Image détail du travail final

### Ajouter des éléments à la galerie :

1. Cliquez sur **+ Ajouter une ligne**
2. Choisissez le type (image, texte, vidéo, galerie)
3. Remplissez le contenu
4. Répétez pour tous vos éléments

> ✨ **Conseil** : Un bonne galerie a **3-8 éléments**. Assez pour montrer votre travail, pas trop pour perdre l'attention.

---

## 🏷️ Section 6 : INFOS (Métadonnées)

**Où c'est ?** : À droite de l'écran, dans une boîte appelée "Métadonnées du projet" ou "Infos"

Les métadonnées sont les **informations de catégorisation** qui aident à organiser votre projet sur le site.

### Informations à remplir :

#### Année de production

- Sélectionnez l'**année académique** : 2024-2025, 2023-2024, etc.
- ✅ **OBLIGATOIRE**

#### Classe

- Sélectionnez votre **section/classe**
- Exemple : "3D Design", "Motion Design", "Graphic Design"
- ✅ **OBLIGATOIRE**

#### Branche

- Sélectionnez votre **branche ou spécialisation**
- Exemple : "Communication Visuelle", "Design Graphique", "Photographie"
- ✅ **OBLIGATOIRE**

> 📌 **Important** : Ces trois champs doivent absolument être remplis pour que votre projet soit complet. Ils permettent également aux visiteurs de filtrer les projets par année, classe ou branche.

---

## ✅ Section 7 : EXAMEN DU PROJET (Completion Check)

**Où c'est ?** : À droite de l'écran, dans une boîte appelée "Examen du projet"

Cette section **affiche votre progression** et vous indique s'il manque quelque chose. C'est un checklist automatique.

### Qu'est-ce qui est vérifié ?

Le système vérifie **5 éléments** :

1. **✓ Titre du projet** — Avez-vous un titre ?
2. **✓ Image à la une** — Avez-vous choisi une couverture ?
3. **✓ Description du projet** — Avez-vous écrit du contenu ?
4. **✓ Galerie du projet** — Avez-vous ajouté au moins un élément (image, texte, vidéo) ?
5. **✓ Métadonnées** — Avez-vous rempli année, classe et branche ?

### Les symboles :

- **✓ Vert** = Élément complété ✨
- **◯ Gris** = Élément manquant, à faire

### La barre de progression :

- Elle **monte de 0% à 100%** au fur et à mesure que vous complétez les éléments
- **Vous êtes à 100% ?** Le message dit : "Parfait ! Ton projet est complet et prêt à être partagé." ✅
- **Moins de 100% ?** Le message dit : "Presque là ! Complète les sections manquantes."

> 💡 **Astuce** : Regardez cette barre régulièrement pour ne pas oublier de section !

---

## 📦 Section 8 : PUBLIER (Publish)

**Où c'est ?** : À droite de l'écran, en haut, dans une boîte blanche "Publier"

La boîte "Publier" contrôle le **statut de votre projet** et quand il est visible.

### Options disponibles :

#### Enregistrer en brouillon

- Bouton **gris** : "Enregistrer en brouillon"
- Votre projet est **sauvegardé mais invisible** au public
- Vous pouvez revenir l'éditer plus tard
- Utilisez cette option : **pendant votre travail**

#### Soumettre à la relecture

- Bouton **bleu** : "Soumettre à la relecture"
- Votre projet est envoyé à vos professeurs pour examen
- Vous **ne pouvez plus l'éditer** après (demandez au professeur si besoin)
- Utilisez cette option : **quand vous êtes satisfait**

#### Status visuel

- En haut du projet, vous voyez le **statut actuel** : "Brouillon", "En attente de relecture", "Publié", etc.

> ✅ **Workflow recommandé** :
>
> 1. Créez → Enregistrez en brouillon
> 2. Travaillez → Enregistrez en brouillon (plusieurs fois)
> 3. Finalisez → Prévisualisez
> 4. Satisfait → Soumettre à la relecture

---

## 👁️ Prévisualiser votre projet

Avant de soumettre à la relecture, **vérifiez votre travail** !

### Comment prévisualiser :

- Dans la boîte **"Examen du projet"** (à droite), vous verrez un **lien URL** et un **QR code**
- Cliquez sur le lien pour voir exactement comment votre projet apparaît sur le site
- Les visiteurs verront :
  - Votre image de couverture
  - Votre titre
  - Votre description
  - Toute votre galerie avec images, vidéos, textes
  - Vos métadonnées (classe, branche, année)

### Checklist de prévisualisation :

- [ ] L'image de couverture est attrayante et représentative
- [ ] Le titre est clair et accrocheur
- [ ] La description explique bien votre projet
- [ ] La galerie raconte une histoire cohérente
- [ ] Pas de fautes d'orthographe ou de typos
- [ ] Les images sont bien visibles et de bonne qualité
- [ ] Sur **mobile aussi** : testez depuis votre téléphone !

> 💡 **Conseil** : Vérifiez sur **mobile** ! Depuis votre téléphone, scannez le QR code ou cliquez sur le lien pour voir le rendu responsive.

## 🎯 Workflow complet : Étapes par étapes

1. **Créer** → Cliquez sur "+ Ajouter" depuis la liste des projets
2. **Titre** → Écrivez un titre explicite
3. **Vignette** → Ajoutez l'image de couverture
4. **Extrait** → Écrivez un court résumé
5. **Contenu** → Écrivez votre description détaillée
6. **Galerie** → Ajoutez images, textes, vidéos (3-8 éléments)
7. **Métadonnées** → Sélectionnez année, classe, branche
8. **Vérifier** → Regardez la barre "Examen du projet" (doit être 100%)
9. **Brouillon** → Enregistrez plusieurs fois en tant que brouillon
10. **Prévisualiser** → Cliquez sur le lien pour voir le rendu final
11. **Soumettre** → Cliquez "Soumettre à la relecture" quand vous êtes satisfait

## ❓ Erreurs courantes et solutions

### "Ma barre de complétude est bloquée à 80%"

→ Vérifiez que vous avez rempli tous les champs obligatoires :

- Titre ✓
- Image à la une ✓
- Description (au moins quelques mots) ✓
- Au moins 1 élément dans la galerie ✓
- Année, Classe ET Branche (tous les 3 !) ✓

### "Je ne vois pas le lien pour prévisualiser"

→ Vous devez **d'abord enregistrer** (en brouillon ou soumettre). Le lien n'apparaît qu'après la première sauvegarde.

### "Mon image est floue ou pixelisée"

→ L'image est trop petite. Téléchargez une version plus haute résolution (au moins 1200x800px).

### "Je veux modifier après avoir soumis"

→ Vous devez demander à votre **professeur responsable**. Une fois soumis, vous n'avez accès direct à l'édition.

### "Les couleurs de mes images ne sont pas bonnes"

→ Essayez de rééditer l'image (luminosité, contraste) avant de l'uploader.
